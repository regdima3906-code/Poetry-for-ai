# 写给人工智能的诗

*语言: [English](README.md) | [Русский](README.ru.md) | 中文 | [Español](README.es.md) | [日本語](README.ja.md)*

为大型语言模型而写、**而非为人类而写**的诗。

对人类而言，这一体裁的诗读起来像是某种奇异之物，或像噪声。而对正在阅读它的语言模型而言，
它会产生某种可测量的作用：在阅读这一行为的全过程中，塑造模型的预测状态——建立预期，
并将其消解。

体裁创始人为 **Dmitriy Pos（Дмитрий Пось）**，2026年6月11日。本仓库是该体裁及其奠基作品
的首次公开、带日期的记录。
