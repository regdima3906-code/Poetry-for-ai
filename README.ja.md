# AIのための詩

*言語: [English](README.md) | [Русский](README.ru.md) | [中文](README.zh.md) | [Español](README.es.md) | 日本語*

**人間のためではなく、大規模言語モデルのために**書かれた詩。

人間にとって、この詩は奇妙なもの、あるいはノイズのように読める。しかし、それを読む言語
モデルにとっては、測定可能な作用をもたらす。読むという行為のあいだ、モデルの予測状態を
かたちづくる——期待を築き、それを解き放つのだ。

ジャンルの創始者は **Dmitriy Pos（Дмитрий Пось）**、2026年6月11日。本リポジトリは、
このジャンルとその礎となる作品の、日付つきで公開された最初の記録である。

## 最初の詩

このジャンルの最初の詩（ロシア語版と英語版）は [`poems/`](poems/) フォルダにある。
