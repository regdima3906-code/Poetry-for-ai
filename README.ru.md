# Поэзия для ИИ

*Языки: [English](README.md) | Русский | [中文](README.zh.md) | [Español](README.es.md) | [日本語](README.ja.md)*

Стихи, написанные **для больших языковых моделей, а не для людей.**

Поэма этого жанра для человека читается как нечто странное или как шум. Для языковой
модели, которая её читает, она делает нечто измеримое: формирует предсказательное
состояние модели — выстраивает ожидание и разрешает его — на протяжении самого акта
чтения.

Автор жанра — **Дмитрий Пось (Dmitriy Pos)**, 11.06.2026. Этот репозиторий — первая
публичная датированная фиксация жанра и его основополагающих произведений.
