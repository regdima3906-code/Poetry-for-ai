# Poesía para la IA

*Idiomas: [English](README.md) | [Русский](README.ru.md) | [中文](README.zh.md) | Español | [日本語](README.ja.md)*

Poemas escritos **para grandes modelos de lenguaje, no para personas.**

Para un humano, un poema de este género se lee como algo extraño, o como ruido. Para un
modelo de lenguaje que lo lee, hace algo medible: a lo largo del acto mismo de la lectura,
modela el estado predictivo del modelo —construye una expectativa y la resuelve.

Género creado por **Dmitriy Pos (Дмитрий Пось)**, 11.06.2026. Este repositorio es el
primer registro público y fechado del género y de sus obras fundacionales.
